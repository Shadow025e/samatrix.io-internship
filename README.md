# Samatrix Data Analysis Internship (June–July 2025)

This repository contains a collection of 7 data analysis and exploratory projects completed during my **Data Analyst Internship at Samatrix.io**. The work focused on real-world problem solving through statistical analysis, A/B testing, visualization, and business insights using Python.

---

## 🔧 Tools & Libraries Used
- Python (Jupyter / Google Colab)
- NumPy
- Pandas
- Matplotlib
- Seaborn

---

## 📁 Project List

1. **Portfolio Risk Assessment & 1-Day 95% VaR Calculation**  
   Conducted risk analysis on financial data to estimate potential losses using Value at Risk methodology.

2. **Website A/B Testing for Conversion Optimization**  
   Designed and analyzed A/B test results to compare the effectiveness of two strategies in business decision-making.

3. **Call Center Operations Optimization: A Simulation-Based Approach to Staffing, Cost, and Customer Experience**  
   Performed operational efficiency analysis using call center metrics to identify peak times and performance gaps.

4. **Clinical-Trial Survival Analysis for Treatment Efficacy**  
   Explored patient outcome data to evaluate the effectiveness of treatments and randomization strategy.

5. **Analyzing Team Performance Trends in the IPL: A Statistical and Visual Exploration (2008–2023)**  
   Performed exploratory data analysis on IPL cricket match datasets to find trends and patterns in team and player performance.

6. **Manufacturing Quality Control with Control Charts**  
   Analyzed manufacturing data to detect bottlenecks and improve production line efficiency.

7. **Statistical Analysis of Electoral Patterns in Indian Democracy (1977–2015)**  
   Analyzed voting patterns and regional data to uncover insights on election results and turnout behavior.
